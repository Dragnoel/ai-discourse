# Cognitive Offloading

Cognitive offloading refers to the process of delegating cognitive work — information storage, computation, reasoning, decision support — to external tools rather than performing it internally. The term covers the use of any external resource to reduce internal cognitive demand, from writing notes to using GPS navigation to querying an AI assistant.

## Offloading in AI-education contexts

In discussions of LLM use in education, cognitive offloading typically carries a specifically negative valence. Vendrell and Johnston (2026) identify it as one of three primary risks of unstructured LLM use (alongside metacognitive disengagement and epistemic narrowing): when students offload to LLMs the effortful cognitive processes — constructing arguments, generating explanations, identifying knowledge gaps — that are essential for deep learning, they bypass the productive struggle that builds durable understanding and critical capacity. This is a qualitatively different concern from efficiency arguments about technology use generally, because LLMs can generate linguistically fluent, superficially plausible outputs that mimic the products of reasoning without the underlying reasoning process.

Bassner et al. (2026) operationalise the same risk in a controlled experiment: ChatGPT conditions allowed students to outsource code-generation, reducing extraneous cognitive load and improving exercise performance, but eliminated the productive cognitive effort required for conceptual learning.

## Desirable vs. undesirable offloading

Not all cognitive offloading is educationally harmful. Offloading genuinely extraneous cognitive burdens (confusing syntax, irrelevant memory retrieval tasks) can free cognitive resources for germane processing — the schema construction that constitutes learning. The educational risk arises when tools offload germane cognitive work, eliminating the very processes that produce understanding. This distinction maps directly onto CLT's germane/extraneous distinction. See [[Cognitive Load]].
